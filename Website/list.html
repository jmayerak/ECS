Unordered List

Home

Let's start by creating an unordered list that will created a list with black circular bullet points. Start with your unordered list tags. Open and close right away. Close tag is 1 line below open. 

    
      <ul> 
      </ul>

Next, you will need to add items by using the list items tags. Add your list item tags. Open and close right away. Do this for as many items as you have. 

    
      <ul>
        <li></li>
        <li></li>
        <li></li>
      </ul>

The above set up would create the following unordered list.

  • Beginning of my list.
  • Middle of my list.
  • End of my list.

Unordered lists have a list-style-type attribute that will change the listing system.

  1. style="list-style-type:disc", default, creates a black filled circle bullet
  2. style="list-style-type:circle", creates an empty circle bullet
  3. style="list-style-type:square", creates a square bullet
  4. style="list-style-type:none", creates unmarked list items
          
  <ul style="list-style-type:square">   
    <li ></li>
    <li ></li>
    <li ></li>
  </ul>
  • Beginning of my list.
  • Middle of my list.
  • End of my list.
 
          
  <ul style="list-style-type:circle" >
    <li></li>
    <li></li>
    <li></li>
  </ul>
  1. Beginning of my list.
  2. Middle of my list.
  3. End of my list.

Ordered List

The steps are exactly the same as above, but we will use ordered list tags this time. Each item will be marked with a number to begin. Start with your ordered list tags. Open and close right away. Close tag is 1 line below open. 

    
      <ol> 
      </ol>

Next, you will need to add items by using the list items tags. Add your list item tags. Open and close right away. Do this for as many items as you have. 

    
      <ol>
        <li></li>
        <li></li>
        <li></li>
      </ol>

The above set up would create the following unordered list.

  1. Beginning of my list.
  2. Middle of my list.
  3. End of my list.

Ordered lists have a type attribute that will change the "numbering" system.

  • type="1" - default, standard numbered list
  • type="A" - uppercase letters
  • type="a" - lowercase letters
  • type="I" - uppercase Roman numerals
  • type="i" - lowercase Roman numerals
          
    <ol type="A">    
      <li></li>
      <li></li>
      <li></li>
    </ol>
  1. Beginning of my list.
  2. Middle of my list.
  3. End of my list.
 
          
    <ol type="I">
      <li></li>
      <li></li>
      <li></li>
    </ol>
  1. Beginning of my list.
  2. Middle of my list.
  3. End of my list.

Nested Lists

Sometimes we would like to have a list of lists, or what we call a nested list. To accomplish this, we simply need to place a list inside of another list. I have provided a generic sample below, and then a large specfic example. 

    
        <ul>
          <li> List Beginning </li>
            <ol>
              <li> Item 1 </li>
              <li> Item 2 </li>
              <li> Item 3 </li>
            </ol>
          <li> List Middle </li>
            <ul>
              <li> Item 1 </li>
              <li> Item 2 </li>
              <li> Item 3 </li>
            </ul>
        </ul>
