--Start with your table tags. Open and close right away. Close tag is 1 line below open.

    
      < table >
      < /table >

Create each table row (horizontal, L to R) of data next. Start with your table row tags. Open and close right away. Close tag is 1 line below open tag. 

    
      < table >
        < tr >
        < /tr >
      < /table >

Then enter your table data tags between your table row tags. Open and close right away ON THE SAME LINE. 

    
      < table >
        < tr >
          < td > < /td >
          < td > < /td >
          < td > < /td >
        < /tr >
      < /table >

The above set up would create the following table design.

This is cell 1 in row 1. This is cell 2 in row 1. This is cell 3 in row 1.

If we continued the table row and table data pattern we could create any style of table that we wanted to.

This is cell 1 in row 1. This is cell 2 in row 1. This is cell 3 in row 1.
This is cell 1 in row 2. This is cell 2 in row 2. This is cell 3 in row 2.
This is cell 1 in row 3. This is cell 2 in row 3. This is cell 3 in row 3.

TABLE WITH HEADERS

--Start with your table tags. Open and close right away. Close tag is 1 line below open.

    
      < table >
      < /table >

Table headers are used to label the columns of the table. Use them inside row tags instead of table data tags. Start with your table row tags. Close tag is 1 line below open tag. Then enter your table header tags. Open and close right away ON THE SAME LINE. This will bold and center your information. 

    
      < table >
        < tr > 
          < th > < /th >
          < th > < /th >
          < th > < /th >
        < /tr >
      < /table >

So now we have the same as above, this time with column headers.

Column 1 Column 2 Column 3
This is cell 1 in row 1. This is cell 2 in row 1. This is cell 3 in row 1.
This is cell 1 in row 2. This is cell 2 in row 2. This is cell 3 in row 2.
This is cell 1 in row 3. This is cell 2 in row 3. This is cell 3 in row 3.

We can also create row headers if we wanted to.

Row 1 This is cell 1 in row 1. This is cell 2 in row 1. This is cell 3 in row 1.
Row 2 This is cell 1 in row 2. This is cell 2 in row 2. This is cell 3 in row 2.
Row 3 This is cell 1 in row 3. This is cell 2 in row 3. This is cell 3 in row 3.

TABLE WITH BORDERS

--Create your table using one of the above methods. From there you need to choose some table style properties to play with. You can apply these to many different tags: table, th, tr, and td. Here is a list of some style properties.

  • Properties:
    • border: #px borderType color;
    • border-collapse: collapse;
    • background-color: color;
    • border-radius: #px; (Cause rounded borders.)
    • border-color: color;

Here is a list of border types that you can enter: dotted, dashed, solid, double, groove, ridge, inset, outset, none, and hidden.

Here is a list of border collapse options that you can enter: separate and collapse.

Here is the HTML code that would create the table at the bottom of the page. 

    
      < head >
        < style >
          table {
            border: 1px solid black;
            border-collapse: initial;
            background-color: black; 
            border-radius: 4px;
            border-color: black;
          }
          tr, th {
            background-color: silver; 
            color: black; 
            border-radius: 4px;
          }
          
          td {
            background-color: plum; 
            color: white; 
            border-radius: 4px;
          }
        < /style>
      < head >
      
      < body >
